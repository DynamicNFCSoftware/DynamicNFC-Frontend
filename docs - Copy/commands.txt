Backend’de yaptığın değişikliklerin React uygulamasında görülmesi için birkaç önemli adımı takip etmen gerekiyor:

1️⃣ Backend’i yeniden derle ve çalıştır

Spring Boot projesinde değişiklik yaptıysan, yeni derlenmiş JAR dosyası gerekiyor.

# Maven kullanıyorsan
mvn clean package


Bu, target/backend-0.0.1-SNAPSHOT.jar dosyasını günceller.

2️⃣ Docker container’ını güncelle

Docker kullanıyorsan, eski image artık değişiklikleri içermiyor. Güncellemek için:

# Docker image’i yeniden build et
docker build -t dynamicnfc-springboot-app:latest backend/

# Çalışan container’ı durdur
docker stop springboot-app
docker rm springboot-app

# Yeni image ile container başlat
docker run -d --name springboot-app -p 8080:8080 dynamicnfc-springboot-app:latest


Eğer docker-compose kullanıyorsan:

docker-compose down
docker-compose up --build -d
--build flag’i, Dockerfile’daki yeni değişiklikleri uygular.

Docker-compose installation
COMPOSE_VERSION=$(curl -s https://api.github.com/repos/docker/compose/releases/latest | grep tag_name | cut -d '"' -f 4)
sudo curl -SL https://github.com/docker/compose/releases/download/${COMPOSE_VERSION}/docker-compose-$(uname -s)-$(uname -m) -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
